## ⚡ The Vision
**ARV-GYM-ARV** is a high-performance training interface designed for elite progression tracking. It combines a futuristic cyber-noir aesthetic with raw functionality, ensuring that every set and every rep is accounted for in your journey to absolute power.

## 🚀 Core Features
* **Multi-User Architecture:** Seamlessly switch between different user profiles. Perfect for training partners or multi-athlete environments.
* **Intelligent Progress Radar:** Instant feedback on your last session's performance, personal records (PR), and RPE-based load suggestions.
* **Bilingual UI Toggle:** Instant switch between **English** and **Polish** interfaces, optimized for high-readability during intense workouts.
* **Integrated Rest Timer:** Custom rest intervals (Strength/Mass/Definition) with haptic feedback to maintain training density.
* **Zero-Latency Storage:** Utilizes `LocalStorage` for persistent data tracking without the need for external databases or logins.

## 🛠️ Technical Specifications
* **Architecture:** Vanilla JavaScript (ES6+ logic)
* **Styling:** CSS3 Modern UI (Flexbox, CSS Grid, Custom Neon Variables)
* **Performance:** Lightweight, zero-dependency codebase for instant mobile loading.
* **Typography:** Orbitron & Google Fonts for a tactical, futuristic feel.
